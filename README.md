# YaMetrika

Ruby gem for [API Yandex Metrika](https://api.yandex.ru/metrika/)

## Installation

Add this line to your application's Gemfile:
```ruby
gem 'ya_metrika'
```
And then execute:
```console
$ bundle
```
Or install it yourself as:
```console
$ gem install ya_metrika
```
